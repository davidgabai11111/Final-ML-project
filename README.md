# Final Machine Learning Project

## Description
Final project for the Machine Learning course.  
The objective is to predict asset performance using interpretable financial features and machine learning models.

## Methods
- Feature engineering (momentum, volatility, reversal)
- Models: Logistic Regression, Random Forest, SVM
- Evaluation metrics: F1-score, Sharpe ratio, CAGR

## Tools
- Python
- Google Colab
- scikit-learn, pandas, numpy
